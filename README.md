# 🍽️ MenuQR

`Python` `Django` `Bootstrap` `HTML` `CSS` `SQLite`

---

## 🎯 Project Objective

**MenuQR** is a **Django-based web application** that enables restaurants to generate and download **QR codes** for their **digital menus**. It allows customers to access menus instantly via smartphone, promoting a contactless, hygienic, and modern dining experience.

---

## 🌐 Live Demo

👉 **[View Live on Render](https://menuqr-zta8.onrender.com/)** 

---

## ✨ Features

- 🔗 Generate QR codes using restaurant name and menu URL  
- ⬇️ Download QR codes for printing and table display  
- 📱 Contactless menu access via smartphone scanning  
- 🎨 Clean and responsive user interface  
- ⚡ Fast and simple QR generation process  
- 🧼 Reduces dependency on physical (and potentially unhygienic) menus
